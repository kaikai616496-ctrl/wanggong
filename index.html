☁️ Google Sheets 同步設定
設定完成後，Windows、Mac、iPhone 打開這個作戰室，資料全部同步。
步驟一｜建立 Google Sheets
1
打開 sheets.google.com，登入你的 Google 帳號
2
點「空白試算表」→ 把試算表命名為「王宮作戰室同步」
3
在 A1 輸入 key，B1 輸入 value，建立兩個欄位標題
步驟二｜建立 Apps Script API
4
在 Google Sheets 上方選單點「擴充功能」→「Apps Script」
5
把原本的程式碼全部刪掉，貼上以下程式碼：
const SS = SpreadsheetApp.getActiveSpreadsheet(); const SH = SS.getSheets()[0]; function doGet(e) { const key = e.parameter.key; const data = SH.getDataRange().getValues(); for (let r of data) { if (r[0] === key) { return ok(r[1]); } } return ok(''); } function doPost(e) { const p = JSON.parse(e.postData.contents); const data = SH.getDataRange().getValues(); for (let i = 0; i < data.length; i++) { if (data[i][0] === p.key) { SH.getRange(i+1,2).setValue(p.value); return ok('updated'); } } SH.appendRow([p.key, p.value]); return ok('created'); } function ok(v) { return ContentService .createTextOutput(JSON.stringify({ok:true,value:v})) .setMimeType(ContentService.MimeType.JSON); }
6
點上方「部署」→「新增部署」→ 類型選「網頁應用程式」
7
「執行身分」選「我」，「誰可以存取」選「所有人」→ 點「部署」→ 複製出現的「網頁應用程式網址」
步驟三｜貼入網址完成連接
8
把複製的網址貼到下方，點「連接」
✅ 連接成功後：每次點「儲存」按鈕，資料自動上傳到 Google Sheets。任何裝置開啟作戰室，自動讀取最新資料。
